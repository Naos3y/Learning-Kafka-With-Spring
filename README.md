# Learning Kafka with Spring
This project is a hands-on exploration of Apache Kafka using Spring Boot. It covers the fundamentals of messaging, including producing and consuming messages in a Kafka topic.

## Technologies Used
**Spring Boot** – For building Kafka producers and consumers

**Apache Kafka** – For event streaming and message processing

**Docker** – To set up and run Kafka in a containerized environment

## Features
- Publish and consume messages using Kafka topics
- JSON message serialization and deserialization
- Configuring Kafka with Spring Boot
- Running Kafka in a Docker container
